# learnstant
学ぶ(learn) + instant = learnstant.com

## setup

```
$ npm install
```


## start

```
$ node app.js
```

access to <http://localhost:3000>

## mongodb

start

```
$ mongod --dbpath db --logpath=mongodb.log &
```

stop

```
$ mongod --shutdown
or 
$ kill -2 
```
